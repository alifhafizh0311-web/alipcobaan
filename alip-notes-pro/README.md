# ALIP-Notes Pro
Skeleton project: Next.js frontend (with NextAuth + Supabase examples) and deployment notes.

Features included:
- Next.js pages + API routes
- Example integration with Supabase (client-side)
- Email auth skeleton (NextAuth config stub)
- README: how to set up Supabase and environment variables

Build & run:
1. cd web
2. npm install
3. cp .env.example .env.local  (fill values)
4. npm run dev
